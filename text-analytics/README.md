## Text Analytics Projects

A collection of NLP projects demonstrating the evolution of text classification models, from statistical approaches to deep learning and transformers.

### Projects

1. **N-grams**: Bigram & trigram models with smoothing and text generation  
2. **MLP**: Feedforward neural network for text classification  
3. **RNN + Attention**: BiLSTM/GRU capturing sequential dependencies  
4. **CNN**: n-gram convolutional filters with pooling and residual connections  
5. **BERT**: Fine-tuned transformer for sentiment classification  

### Summary

Progression from **simple statistical models → neural networks → transformers**, highlighting increasing capability to capture context and semantics in text.
