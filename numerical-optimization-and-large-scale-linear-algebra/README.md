## Numerical Optimization & Large-Scale Linear Algebra

Collection of projects focused on numerical methods, linear algebra, and large-scale data analysis.

### Projects



**1. Lamp Power Optimization (Least Squares)**  
- Modeled illumination as a linear system  
- Solved least squares problem for optimal lamp intensities  
- Applied constraints (non-negativity, energy limits)  
- Visualized illumination patterns and errors  
- Evaluated performance using RMS error

**2. SVD-based Digit Classification**  
- Class-wise SVD decomposition  
- Low-rank representations (5–20 components)  
- Classification via least squares residuals  
- Analysis of accuracy vs basis size  

**3. PageRank Analysis**  
- Power Method for eigenvector computation  
- Gauss–Seidel for linear system formulation  
- Convergence comparison and performance analysis  
- Impact of damping factor (α) on ranking stability  

 
### Summary

Focus on core numerical techniques for large-scale problems, including matrix factorization, iterative methods, and optimization.  
Emphasis on convergence behavior, computational efficiency, and real-world modeling.
